# Bài A – So sánh hai số

## Đề bài
Nhập vào hai số nguyên a và b. Hãy cho biết số nào lớn hơn.

## Input
Hai số nguyên a và b (trên một dòng).

## Output
Chuỗi "a lớn hơn b" hoặc "b lớn hơn a" hoặc "a bằng b".

## Ví dụ
### Input
```
5 3
```

### Output
```
a lớn hơn b
```

### Input
```
2 8
```

### Output
```
b lớn hơn a
```

### Input
```
7 7
```

### Output
```
a bằng b
```
