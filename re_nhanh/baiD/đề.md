# Bài D – Tìm số lớn nhất trong ba số

## Đề bài
Nhập vào ba số nguyên a, b, c. Hãy in ra số lớn nhất.

## Input
Ba số nguyên a b c (trên một dòng).

## Output
Một số nguyên là số lớn nhất.

## Ví dụ
### Input
```
5 3 8
```

### Output
```
8
```

### Input
```
10 15 7
```

### Output
```
15
```

### Input
```
4 4 4
```

### Output
```
4
```
