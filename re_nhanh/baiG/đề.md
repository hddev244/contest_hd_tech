# Bài G – Phân loại số nguyên

## Đề bài
Nhập vào một số nguyên n. Hãy cho biết n là số dương, số âm hay bằng 0.

## Input
Một số nguyên n.

## Output
Chuỗi "Số dương" hoặc "Số âm" hoặc "Bằng 0".

## Ví dụ
### Input
```
5
```

### Output
```
Số dương
```

### Input
```
-3
```

### Output
```
Số âm
```

### Input
```
0
```

### Output
```
Bằng 0
```
