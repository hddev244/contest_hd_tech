# Bài J – Xếp hạng vận động viên

## Đề bài
Nhập vào số điểm của vận động viên. Xếp hạng như sau:
- Hạng A: điểm >= 90
- Hạng B: 75 <= điểm < 90
- Hạng C: 60 <= điểm < 75
- Hạng D: điểm < 60

## Input
Một số nguyên là điểm.

## Output
Chuỗi "Hạng A", "Hạng B", "Hạng C" hoặc "Hạng D".

## Ví dụ
### Input
```
95
```

### Output
```
Hạng A
```

### Input
```
80
```

### Output
```
Hạng B
```

### Input
```
65
```

### Output
```
Hạng C
```

### Input
```
50
```

### Output
```
Hạng D
```
