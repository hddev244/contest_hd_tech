# Bài F – Kiểm tra tuổi đi bầu cử

## Đề bài
Nhập vào tuổi của một người. Nếu tuổi >= 18 thì in "Đủ điều kiện", ngược lại in "Chưa đủ điều kiện".

## Input
Một số nguyên là tuổi.

## Output
Chuỗi "Đủ điều kiện" hoặc "Chưa đủ điều kiện".

## Ví dụ
### Input
```
20
```

### Output
```
Đủ điều kiện
```

### Input
```
16
```

### Output
```
Chưa đủ điều kiện
```

### Input
```
18
```

### Output
```
Đủ điều kiện
```
