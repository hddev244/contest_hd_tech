# Bài C – Xếp loại điểm

## Đề bài
Nhập vào điểm số (0–10). Hãy in ra xếp loại:
- "Giỏi" nếu điểm >= 8
- "Khá" nếu 6.5 <= điểm < 8
- "Trung bình" nếu 5 <= điểm < 6.5
- "Yếu" nếu điểm < 5

## Input
Một số thực là điểm.

## Output
Chuỗi xếp loại tương ứng.

## Ví dụ
### Input
```
8.5
```

### Output
```
Giỏi
```

### Input
```
7.0
```

### Output
```
Khá
```

### Input
```
5.5
```

### Output
```
Trung bình
```

### Input
```
3.5
```

### Output
```
Yếu
```
